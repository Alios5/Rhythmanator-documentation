# Interface

Discover the powerful interface of RhythmNator Editor.

## Home Screen

The home screen is your starting point. It allows you to manage your projects and customize your environment via the top toolbar (themes, shortcuts, fonts, languages).

[See home interface details](./interface-home.md)


## The Editor

The main workspace where the magic happens.

### 1. Audio Waveform
Located at the top, it shows the visual representation of your audio. Use it for precise navigation.

### 2. Multi-track Timeline
The central part of the editor.
- **Grids**: Visual beat and measure lines.
- **Snapping**: Enable grid snapping for perfect timing.
- **Zoom**: Zoom in for detailed mapping or out for an overview.
- **Smart Groups**: Organize your tracks and notes with powerful grouping logic.

### 3. Toolbar
Quick access to selection tools, note placement, and playback commands.

### 4. Note Actions and Export
- **Custom Actions**: Select any note to assign it gameplay logic or specific actions in the properties panel.
- **JSON Export**: Export your sheets in a clean JSON format, ready for production.
