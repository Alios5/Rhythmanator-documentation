# Guide: Project Launch

There are three different ways to launch a project in RhythmNator Editor. This guide explains how to use each of them.

---

## 1. Create a New Project

The most common method to start a new sheet.

1. Click the **"New Project"** button.
2. Fill in the three fields in the form:
    - **Project Name**: The title of your creation.
    - **Folder**: The save location on your computer.
    - **Music** (Optional): The audio file to map.
3. Click **"Create"** to launch the editor.

## 2. Open an Existing Project

If you already have a project file on your disk.

1. Click the **"Open a Project"** button located just below the first button.
2. Navigate your folders via the system dialog.
3. Select your file with the **`.rne`** extension.
4. The project will load and open automatically.

## 3. Use the Recent Projects List

To quickly pick up where you left off.

1. Check the **Recent Projects** list on the home screen.
2. Click directly on the card of the project you recently worked on.
3. The editor will open immediately with that project.

## File Format

Projects are saved in **`.rne`** format. These are lightweight JSON-based files that store note positions and metadata. Always use `Ctrl+S` in the editor to manually save your progress.
