# Guide: Note

Learn how to manipulate notes, the central element of your rhythm.

## Note Manipulation

To create and edit notes, use the tools in the upper tracks bar.

### Edit Mode (Pencil)

Activate the **Edit** tool to place notes manually.

-   **Create a note**: **Left-click** on a track at the desired location.
-   **Long note (Sustain)**: **Hold Left-click + drag** to extend the note's duration.
-   **Delete**: **Right-click** on a note to erase it.
-   **Multiple deletion**: **Hold Right-click + drag** to erase several notes in a row ("eraser").

### Live Recording

You can create notes in real-time while listening:

1.  Start the music **playback**.
2.  Press the **assigned key** for the corresponding track at the right moment.
3.  Notes will automatically appear synchronized with your keystroke.

### Selection Mode (Cursor)

Activate the **Selection** tool to manipulate groups of notes.

-   **Multiple selection**: Draw a frame around the notes you want to select.
-   **Move**: Click and drag the selection to change its time or track.
-   **Copy / Paste / Duplicate**: Use the usual shortcuts or the context menu on the selection.
-   **Merge**: Select several notes on the same track and choose **Merge** to combine them into a single continuous long note.
