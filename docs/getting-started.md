# Getting Started

Welcome to RhythmNator Editor! This guide will help you install the tool and create your first rhythmic mapping project.

## Step 1: Tool Installation

RhythmNator Editor is currently available on Windows.

### Platform Availability
- **Windows**: Available. Download the tool and launch the installer to get started.
- **macOS**: Coming soon.
- **Linux**: Coming soon.

---

## Step 2: Create a New Project

Once the tool is installed, follow these steps to launch your first creation:

1. **Launch the tool**: Open the RhythmNator Editor app.
2. **Project Area**: In the home interface, go to the dedicated project area.
3. **New Project**: Click the **"New Project"** button.
4. **Project Configuration**: Fill in the required information in the form:
    - **Project Name**: Choose a name for your sheet.
    - **Folder**: Select the location where the project will be saved.
    - **Music**: Load your audio file (MP3, WAV, or OGG).
