# Overview

RhythmNator Editor is a desktop application designed for rhythm game creators. It offers a visual and intuitive way to synchronize notes with music.

## What is RhythmNator Editor?

It allows you to:
- Load any audio file and visualize its waveform with **Sample-Accurate Timing**.
- Automatically detect BPM and synchronize with **Pitch Preservation** during playback.
- Place notes on a **Multi-track Timeline** with **Smart Groups**.
- Organize your project with themes and groups.
- Export your work in lightweight `.rne` files or **Clean JSON**.
