# Guide : Zone de Travail (Piste)

Apprenez à gérer les pistes et organiser votre partition dans la "Sheet".

## Création d'une Piste

1.  Allez dans le **Panneau de gestion des pistes**.
2.  Cliquez sur le bouton **Créer une nouvelle piste**.
3.  Une boîte de dialogue s'affiche :
    -   Entrez le **Nom** de la nouvelle piste.
    -   Choisissez la **Touche** (clavier) qui sera utilisée pour placer les notes.
    -   Choisissez la **Couleur** des notes pour cette piste.

## Options de Piste

En haut à gauche de chaque piste se trouve un **Label**. Cliquez dessus pour ouvrir le menu d'options :

-   **Modifier** : Permet de changer les paramètres (nom, touche, couleur).
-   **Cacher la piste** : Masque la piste pour clarifier la vue.
-   **Supprimer la piste** : Supprime la piste et ses notes.

*(L'assignation aux groupes et la manipulation des notes sont couvertes dans leurs guides respectifs : [Guide : Groupes](./guide-groups.md) et [Guide : Note](./guide-event-keys.md))*

## Navigation

Maîtrisez les déplacements dans votre projet pour éditer avec précision.

### Le Lecteur (Playback)

Situé en bas de l'écran, le lecteur affiche la durée totale et la position actuelle.

-   **Lecture / Pause** : Lance ou arrête la musique.
-   **Stop** : Arrête et revient au début ou au dernier point de départ.
-   **Sauter +/- 10s** : Avance ou recule rapidement de 10 secondes.
-   **Barre de Lecture** : Glissez le curseur pour atteindre un moment précis.

### Visualiseur et Zone de Travail

-   **Clic sur le Visualiseur** : Cliquez n'importe où sur la forme d'onde (en haut) pour placer instantanément la tête de lecture à cet endroit.
-   **Ligne de Lecture (Playhead)** : La ligne verticale indique votre position exacte dans la musique.
-   **Barres de Défilement (Scrollbars)** : Utilisez les barres horizontale et verticale pour naviguer visuellement dans la zone des pistes.

### Suivi Automatique (Auto-Follow)

Le bouton tout à gauche du lecteur active le mode **Suivi**.

-   **Activé** : La vue défile automatiquement pour garder la ligne de lecture visible.
-   **Désactivé** : La ligne de lecture peut sortir de l'écran, vous permettant de travailler sur une autre section pendant l'écoute.
