# Guide : Gestion de l'Audio et BPM

Une synchronisation parfaite entre la musique et votre grille est la clé d'une bonne partition. Ce guide vous explique comment gérer vos fichiers audio et configurer le tempo (BPM).

---

## 1. Gestion de la Musique

Dans RhythmNator, vous pouvez charger ou changer de musique à tout moment durant votre projet.

> [!WARNING]  
> Si vous chargez une nouvelle musique plus courte que la précédente, toutes les notes situées au-delà de la nouvelle durée seront **automatiquement supprimées**.

---

## 2. Configuration du BPM

Le BPM (Battements Par Minute) définit la vitesse de la grille rythmique.

- **Automatique** : Cliquez sur l'icône **Éclair** dans le panneau audio pour que l'éditeur tente de détecter le BPM tout seul.
- **Manuel** : Vous pouvez saisir directement la valeur si vous la connaissez.
- **Analyse externe** : Si le BPM trouvé n'est pas correct, nous vous conseillons d'utiliser des sites tiers d'analyse de BPM pour obtenir une valeur précise.

## 3. Décalage de départ (Offset)

L'offset permet de décaler toute la grille pour l'aligner parfaitement sur le premier battement de la musique. C'est l'étape cruciale pour que vos notes ne soient pas "à côté" du rythme.

## 4. Synchronisation du Rythme et Lecture

Plus bas dans le panneau, vous trouverez des options de synchronisation avancées :

- **Signature Rythmique** : Choisissez le type de rythme adapté à votre musique (ex: 4/4).
- **Volume** : Ajustez le niveau sonore de l'aperçu.
- **Vitesse (Pitch)** : Accélérez ou ralentissez la lecture pour placer vos notes avec une précision chirurgicale.
