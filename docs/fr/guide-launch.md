# Guide : Lancement de projet

Il existe trois manières différentes de lancer un projet dans RhythmNator Editor. Ce guide vous explique comment utiliser chacune d'entre elles.

---

## 1. Créer un nouveau projet

La méthode la plus courante pour débuter une nouvelle partition.

1. Cliquez sur le bouton **"Nouveau projet"**.
2. Remplissez les trois champs du formulaire :
    - **Nom du projet** : Le titre de votre création.
    - **Dossier** : L'emplacement de sauvegarde sur votre ordinateur.
    - **Musique** (Optionnel) : Le fichier audio à mapper.
3. Cliquez sur **"Créer"** pour lancer l'éditeur.

## 2. Ouvrir un projet existant

Si vous avez déjà un fichier de projet sur votre disque.

1. Cliquez sur le bouton **"Ouvrir un projet"** situé juste en bas du premier bouton.
2. Naviguez dans vos dossiers via le dialogue système.
3. Sélectionnez votre fichier avec l'extension **`.rne`**.
4. Le projet se chargera et s'ouvrira automatiquement.

## 3. Utiliser la liste des projets récents

Pour reprendre rapidement là où vous vous étiez arrêté.

1. Consultez la liste des **Projets récents** sur l'écran d'accueil.
2. Cliquez directement sur la carte du projet sur lequel vous avez travaillé récemment.
3. L'éditeur s'ouvrira immédiatement avec ce projet.

## Format de fichier
Les projets sont sauvegardés au format **`.rne`**. Ce sont des fichiers légers basés sur JSON qui stockent les positions des notes et les métadonnées. Utilisez toujours `Ctrl+S` dans l'éditeur pour sauvegarder manuellement votre progression.
