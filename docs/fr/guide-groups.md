# Guide : Groupes

Organisez votre chronologie avec les Groupes Intelligents (Smart Groups).

## Utilisation des groupes
Les groupes vous permettent de rassembler plusieurs pistes. C'est utile pour séparer les différentes sections d'une chanson ou différents types d'instruments (par exemple, toutes les pistes de percussion dans un seul groupe).

## Gestion des Groupes

Les **Groupes** vous permettent d'organiser vos pistes, ce qui est particulièrement utile pour les projets complexes avec de nombreux instruments.

### Créer un Groupe

1.  Dans le **Panneau de gestion des pistes**, cliquez sur le bouton **Créer un Groupe**.
2.  Une boîte de dialogue s'affiche :
    -   Entrez le **Nom du Groupe** (ex: Batterie, Mélodie).
3.  Le nouveau groupe apparaît dans la liste.

### Gérer les Pistes dans un Groupe

-   **Ajouter** : Cliquez sur le bouton **(+)** à côté du nom du groupe pour y déplacer des pistes existantes.
-   **Retirer** : Cliquez sur le bouton **(-)** à côté d'une piste *dans* un groupe pour la sortir de celui-ci.

### Options de Groupe

Tout comme les pistes individuelles, les groupes disposent d'options :

-   **Modifier** : Renommer le groupe.
-   **Supprimer** : Supprimer le groupe (les pistes à l'intérieur seront sorties ou supprimées selon votre choix).
-   **Cacher (Bouton Œil)** : Cliquez sur l'œil à gauche du nom du groupe pour **Muter/Cacher** toutes les pistes qu'il contient d'un seul coup. C'est idéal pour se concentrer sur une partie spécifique de la musique.

## Logique et Hiérarchie
- **Visibilité** : Activez ou désactivez la visibilité du groupe pour épurer votre espace de travail.
- **Actions groupées** : Appliquez des modifications à toutes les pistes d'un groupe simultanément.
- **Export** : Les groupes sont préservés dans l'export JSON pour une organisation logique dans votre jeu.
