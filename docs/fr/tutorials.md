# Tutoriels

Maîtrisez RhythmNator Editor avec ces guides étape par étape.

## Mapping de base

1. **Importer de l'audio** : Commencez par charger votre piste préférée.
2. **Détecter le BPM** : Utilisez la détection automatique ou réglez-le manuellement pour aligner la grille.
3. **Placer des notes** : Sélectionnez une piste et cliquez sur la chronologie pour placer des notes.
4. **Lecture** : Utilisez la barre d'espace pour lire/mettre en pause et écouter la synchronisation.

## Techniques Avancées

### Travailler avec des calques
Utilisez plusieurs pistes pour séparer différents types de notes (ex: voix, batterie, mélodies).

### Personnalisation des notes
Vous pouvez faire glisser les extrémités des notes pour créer des notes longues (sustains) ou modifier leurs propriétés pour déclencher des effets de jeu spéciaux.

### Utiliser les raccourcis
Apprenez les principaux raccourcis :
- `Q/W/E/R` : Changer d'outil.
- `S` : Basculer l'encliquetage.
- `Z/X` : Changer la division de la grille.
- `?` : Voir tous les raccourcis.
