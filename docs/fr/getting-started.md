# Premiers pas

Bienvenue dans RhythmNator Editor ! Ce guide vous aidera à installer l'outil et à créer votre premier projet de mapping rythmique.

## Étape 1 : Installation de l'outil

RhythmNator Editor est actuellement disponible sur Windows.

### Disponibilité par plateforme
- **Windows** : Disponible. Téléchargez l'outil et lancez l'installateur pour commencer.
- **macOS** : Bientôt disponible.
- **Linux** : Bientôt disponible.

---

## Étape 2 : Créer un nouveau projet

Une fois l'outil installé, suivez ces étapes pour lancer votre première création :

1. **Lancer l'outil** : Ouvrez l'application RhythmNator Editor.
2. **Zone des projets** : Dans l'interface d'accueil, allez dans la zone dédiée aux projets.
3. **Nouveau Projet** : Cliquez sur le bouton **"Nouveau projet"**.
4. **Configuration du projet** : Remplissez les informations nécessaires dans le formulaire :
    - **Nom du projet** : Choisissez un nom pour votre partition.
    - **Dossier** : Sélectionnez l'emplacement où le projet sera enregistré.
    - **Musique** : Chargez votre fichier audio (MP3, WAV ou OGG).
