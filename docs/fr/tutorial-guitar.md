# Tutoriel : Créer un jeu type Guitar

Apprenez à utiliser RhythmNator Editor pour créer des partitions pour un jeu de rythme basé sur la guitare.

## Étape 1 : Pistes (Lanes)
Créez 5 pistes, chacune représentant une "corde" ou un bouton sur la guitare.

## Étape 2 : Placement des notes
Placez des notes qui correspondent à la mélodie de la guitare. Utilisez les "Sustains" pour les notes tenues et les sections de vibrato.

## Étape 3 : Mapping d'événements
Attribuez des couleurs à chaque piste à l'aide des touches d'événement pour que votre moteur de jeu sache quel bouton afficher à l'écran.

## Étape 4 : Test
Lisez l'audio à 50% de la vitesse pour vous assurer que chaque note s'aligne parfaitement avec les riffs de guitare.
